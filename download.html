HOW TO BUILD YOUR OWN RECOVERY

Requirements

  1. PC with Linux Distro at least 16gb ram
  2. Device Tree
  3. Brain

Setup build environment

pkg install git && git clone http://github.com/akhilnarang/scripts tools && bash tools/setup/android_build_env.sh

Create Folder

mkdir foldernamecd foldername

Cloning TWRP manifest

To initialize a shallow clone, which will save even more space, use a command like this: check manifest twrp for your device
https://github.com/minimal-manifest-twrp

repo init --depth=1 -u https://github.com/minimal-manifest-twrp/platform_manifest_twrp_omni.git -b twrp-10.0-deprecated

Syncing TWRP source

repo sync

CLoning your device tree

Let's clone our device tree in foldername/device/brand/devicecodename üëá

git clone devicetreegithublink device/xiaomi/chopin

Let's start building

. build/envsetup.sh && export ALLOW_MISSING_DEPENDENCIES=true && lunch omni_devicecodename-eng
